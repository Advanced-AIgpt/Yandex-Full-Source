Код для https://st.yandex-team.ru/DIALOG-918  "Исследовать external_skill болталки"

Воркфлоу:
* fill_users_lifetime.ipynb - заполняет справочник по первому и последнему появлению юзера
  Имеет смысл запускать только если нас интересуют даты, ещё не покрытые этим словарём
* log_sampling.ipynb - выборка по пользователям (обрабатывать лог целиком будет слишком дорого из-за производительности ручки /qa/nlu/pa/)
  Перед запуском нужно выбрать регексп для uuid, чтобы получилась выборка нужного размера
* update_intents.ipynb - выгружает в unknown_utterances.tsv предложения, которые пока ещё не встречаются в possible_intents.tsv
  Этот файл нужно натравливать на локальный скрипт try_intent.py и загружать possible_intents.tsv обратно на сервер
* group_to_session.ipynb - делает из таблички с предложениями табличку с сессиями
